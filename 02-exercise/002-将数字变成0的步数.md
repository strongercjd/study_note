# 题目 
给你一个非负整数 num ，请你返回将它变成 0 所需要的步数。 如果当前数字是偶数，你需要把它除以 2 ；否则，减去 1 。

输入：num = 14

输出：6

解释：
1. 14 是偶数，除以 2 得到 7 。
2. 7 是奇数，减 1 得到 6 。
3. 6 是偶数，除以 2 得到 3 。
4. 3 是奇数，减 1 得到 2 。
5. 2 是偶数，除以 2 得到 1 。
6. 1 是奇数，减 1 得到 0 。


# 答案

``` cpp
class Solution {
public:
    int numberOfSteps(int num) {
        unsigned char step_num;
        step_num = 0;
        for(;;)
        {
            if(num%2==0)
            {
                num = num/2;
            }else{
                num = num -1;
            }
            step_num ++;
            if(num == 0)
                return step_num;
        }
    }
};
```

# 推荐答案

``` cpp
class Solution {
public:
    int numberOfSteps(int num) {
        int ret = 0;
        while (num) {
            ret += (num > 1 ? 1 : 0) + (num & 0x01);
            num >>= 1;
        }
        return ret;
    }
};
```

主要运用了

- 第0位为1表示奇数，为0表示偶数

- 向右移动1位表示处以2
