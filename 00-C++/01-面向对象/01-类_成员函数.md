# C++ 类成员函数

## 一、定义与特性

类的**成员函数**是定义在类作用域内的函数，属于类的成员，可直接访问类的所有成员变量（public/protected/private）。

- 在**类定义内部直接实现**的成员函数，编译器会默认将其声明为**内联函数**（无需显式写 `inline` 关键字）。  
- 成员函数的调用必须依附于**类的实例对象**，不能独立调用。

---

## 二、两种定义方式  

### 1. 类内定义（隐式内联）

函数声明与实现直接写在类体中，简洁直观，适合简短函数。

```cpp
class Box
{
public:
    double length;   // 成员变量
    double breadth;
    double height;

    // 类内定义的成员函数
    double getVolume()
    {
        return length * breadth * height;
    }
};
```

### 2. 类外定义（必须用作用域解析运算符 `::`）

函数声明在类内，实现写在类外，必须通过 `类名::函数名` 标识归属，适合复杂函数。

```cpp
class Box
{
public:
    double length;
    double breadth;
    double height;
    double getVolume(); // 仅声明
};

// 类外实现：必须指定所属类（Box::）
double Box::getVolume()
{
    return length * breadth * height;
}
```

---

## 三、调用方式

成员函数必须通过**对象**或**对象指针**调用：

```cpp
Box myBox;          // 创建类实例对象
myBox.getVolume();  // 对象调用：. 运算符

Box* pBox = &myBox;
pBox->getVolume(); // 指针调用：-> 运算符
```

---

## 四、核心底层：隐式 `this` 指针

C++ 非静态成员函数的**第一个参数是编译器自动隐式添加的 `this` 指针**：

- `this` 指针指向**调用该函数的当前对象**，用于区分不同对象的成员变量；
- 该参数由编译器自动处理，**开发者无法手动声明/修改**；
- const 成员函数的 `this` 指针为 `const` 类型。

底层等效逻辑（编译器视角，不可手动编写）：

```cpp
class Box
{
public:
    double length;
    double breadth;
    double height;

    // 编译器隐式添加 this 指针作为第一个参数
    double getVolume(Box* this)
    {
        return this->length * this->breadth * this->height;
    }
};
```

---

## 五、结合 `std::bind` 绑定成员函数

`std::bind` 绑定**非静态成员函数**时，必须手动绑定隐式的 `this` 指针（即指定调用对象）。

### 关键规则

`std::bind` 对成员函数做了**自动兼容处理**：

1. 传入**对象** → `bind` 自动取地址，转换为 `this` 指针；
2. 传入**对象指针** → `bind` 直接使用该指针作为 `this`；
3. 两种写法最终效果完全一致。

### 代码示例

```cpp
Box myBox;

// 写法1：传入对象（bind 自动取地址 &myBox）
auto func1 = std::bind(&Box::getVolume, myBox);

// 写法2：直接传入对象指针
auto func2 = std::bind(&Box::getVolume, &myBox);

// 调用效果完全相同
func1(); 
func2();
```
