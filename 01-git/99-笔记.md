或者使用下面的方法

通过 `git log <远端分支>..<本地分支>` 列出**本地有但远端没有**的提交（即未推送的提交）：
```bash
git log --oneline origin/main..main
```
**输出解读**：

- ✅ 已推送：无任何输出（本地和远端提交完全一致）
- ❌ 未推送：列出未推送的提交哈希和备注（示例）：
    
    plaintext
    
    ```plaintext
    a1b2c3d feat: 新增登录功能
    d4e5f6g fix: 修复表单验证bug
    ```
    
    （`--oneline` 让输出更简洁，去掉则显示完整提交信息）
