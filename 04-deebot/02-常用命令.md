删除配置文件

```bash
dd if=/dev/zero of=/dev/block/bootdevice/by-name/sys bs=512 seek=2178 count=2
```

查看flash中配置文件数据

```bash
hexdump -C -s 0x00110400 -n 1024 /dev/block/bootdevice/by-name/sys
```
