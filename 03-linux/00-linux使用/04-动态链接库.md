# 动态链接库

动态链接库加载时的内存情况

结论:对于代码（.text段），确实只存一份，大家共享；但对于可修改的数据（.data、.bss段），每个进程要么有自己的独立副本，要么通过“写时复制（COW）”隔离。

## 核心机制

共享库（.so）内部结构不同，处理方式也不同：代码段公有只读，数据段私有

代码段 (.text) —— 真正的共享

- **属性**：只读、可执行。
- **内容**：函数逻辑（比如 `foo_add()` 的机器指令）。
- **共享方式**：物理内存中只有**一份**。进程A调用 `foo_add()`，CPU通过页表找到这份唯一的指令去执行。进程B调用时也一样。
- **读取影响**：既然是只读的，任何进程都只是“看”指令，不会修改。所以**完全不影响**。这就好比三个人看同一块路牌，谁也不会把路牌上的字改掉。

数据段 (.data, .bss) —— 看起来共享，实则私有(改变的才会私有)

如果libfoo.so里有一个全局变量 `int g_counter = 10;`，那怎么办？

**如果物理内存中真的只有一份 `g_counter`**，那进程A把它改成20，进程B再读取时就会读到20。这确实是严重的安全和逻辑问题。

解决方案：**写时复制**(Copy-on-Write, COW)

1. **初始状态**：物理内存中仍然只保存一份数据段（包含 `g_counter=10`）。所有进程的虚拟地址空间都指向这一份。此时是共享的、只读的。
2. **第一次写入时**：进程A执行 `g_counter = 20;`。
    - CPU会检测到这是一个**写入**操作，而这块内存页当前是**只读**的（因为要共享）。
    - 硬件触发一个**页故障**，操作系统接管。
    - OS看到：“哦，进程A想写一个COW页。”于是，OS在物理内存中**新分配一页**，把原来的数据（`g_counter=10`）**复制**到新页里。
    - 然后，OS把进程A的页表里指向这个数据段的虚拟地址，**改到这张新页**上，并设置为**可读写**。
    - 最后，重新执行 `g_counter = 20;`，现在写的就是进程A私有的新页了。
3. **结果**：
    - **进程A**：有了自己私有的 `g_counter=20`。
    - **进程B、C**：仍然指向原来的只读页，看到的是 `g_counter=10`。
    - **物理内存**：现在有两份数据段的页了（一份原始只读的给B、C，一份复制后可写的给A）。

## procrank 工具

以下是这四个指标的详细解读：
.
procrank工具使用 VSS、RSS、PSS 和 USS 四个指标来评估进程的内存占用。它们从不同角度描述了进程对内存的“占用”，共同遵循 **VSS ≥ RSS ≥ PSS ≥ USS** 的规律。

| 指标 | 全称 | 关注的核心问题 | 计算方式 / 特点 |
| :--- | :--- | :--- | :--- |
| **VSS** | Virtual Set Size | “进程最多可能占用多少内存？” | **虚拟内存大小**，包含已分配但未实际驻留在物理内存中的部分（如 `malloc` 分配但未写入的空间）。参考价值有限。 |
| **RSS** | Resident Set Size | “物理内存里有多少是给了这个进程？” | **实际驻留的物理内存大小**，包含了该进程使用的**所有共享库**。可能会重复计算共享内存，导致数值偏高。 |
| **PSS** | Proportional Set Size | “按比例分摊后，这个进程实际占用了多少物理内存？” | **按比例分摊的物理内存大小**。将共享内存按使用进程数均摊后计算到每个进程上。是评估进程物理内存“实际消耗”最准确的指标。 |
| **USS** | Unique Set Size | “如果杀掉这个进程，能立刻释放多少物理内存？” | **进程独占的私有内存大小**，不包含任何共享内存。非常适合定位**内存泄漏**问题，因为它代表了终止进程可回收的真实内存量。 |

> Resident：居住的；常驻的；定居的
> Proportional：成比例的；相称的；均衡的
> Unique： 独一无二的；独特的；罕见的

包含哪些内存的对比表

| 指标 | 动态库代码段 | 动态库只读数据段（未修改） | 动态库私有数据段（COW后） | 堆（已写入） | 堆（已申请未写入） | 栈（已使用） |
|------|-------------|--------------------------|--------------------------|-------------|------------------|-------------|
| VSS  | 计入虚拟     | 计入虚拟                 | 计入虚拟                 | 计入虚拟     | 计入虚拟          | 计入虚拟     |
| RSS  | 计入物理（共享） | 计入物理（共享）       | 计入物理（私有）         | 计入物理     | 不计入            | 计入物理     |
| PSS  | 平分物理     | 平分物理                 | 独占计入                 | 独占计入     | 不计入            | 独占计入     |
| USS  | 不计入       | 不计入                   | 独占计入                 | 独占计入     | 不计入            | 独占计入     |
